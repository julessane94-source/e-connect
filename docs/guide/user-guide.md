## 1. Connexion
1. Rendez-vous sur la page de connexion
2. Entrez votre email et mot de passe
3. Cliquez sur "Se connecter"

## 2. Tableau de bord
Le tableau de bord affiche :
- Statistiques des actes
- Demandes récentes
- Tâches en cours
- Notifications

## 3. Gestion de l'état civil

### Mariages
